# Backend Directory

## 目的
本目录用于放项目后续的后端实现。

后端在本项目中的职责是：
- 接收前端请求
- 生成今日 `day plan`
- 聚合 `shopping list`
- 计算 `day_plan_summary`
- 提供历史记录读取
- 统一管理推荐逻辑与数据读取

## 为什么现在还是空的
当前项目仍在需求澄清阶段。

在以下内容完全稳定前，不应急着展开完整后端实现：
- 产品范围
- Core Rules
- 数据模型
- 推荐逻辑
- 内容边界
